# Adventure Works Sales Analysis Dashboard (Excel)

## 📊 Project Overview

This project showcases an interactive sales analysis dashboard built using **Microsoft Excel** based on the Adventure Works dataset. It provides meaningful and interactive insights into sales performance, product profitability, customer behavior, and time-based trends.

---

## 🚀 Key Features

### 🔹 Product Analysis

* Top 5 most profitable products
* Profit distribution by product color
* Product pricing segmentation (Above $150 vs Below $150)

### 🔹 Customer Insights

* Top 5 customers by profit contribution
* Total customers and average customer metrics
* Profit contribution by age group and gender

### 🔹 Sales & Profit Analysis

* Total Revenue, Total Profit, and COGS
* Profit margin analysis
* Transaction overview

### 🔹 Time Analysis

* Monthly profit trends
* Year-wise filtering (2005–2008)
* Weekday vs weekend performance
* Quarterly profit breakdown

### 🔹 Regional Insights

* Country-wise profit contribution
* Map-based visualization of sales performance

---

## 📈 Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers & Filters
* Data Cleaning & Transformation

---

## 📌 Key Insights

* Products priced above $150 contribute the majority of profit (~95%)
* Top 5 products generate a significant share of total profit
* Weekdays contribute more profit compared to weekends
* United States and Australia are major contributors to total profit
